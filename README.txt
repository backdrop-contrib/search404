DESCRIPTION:
------------
Instead of showing a standard "404 Page not found", this module
performs a search on the keywords in the URL.

INSTALLATION:
-------------
1. Copy search404.module into your 'modules' directory.
2. Enable the module at 'administer >> modules'.
3. Check "Set Search 404 to be your default 404 page." at
   admin/settings/search404 and save changes.
