DESCRIPTION:
------------
Instead of showing a standard "404 Page not found", this module
performs a search on the keywords in the URL.

INSTALLATION:
-------------
1. Copy search404.module into your 'modules' directory.
2. Enable the module at 'administer >> modules'.
3. The module will automaticly replace the path to your 404 page with "search404"
UNINSTALLTION:
--------------
1. Disable the module.
2. Uninstall the module, which will set the 404 page
   the path it was before install of this module.
